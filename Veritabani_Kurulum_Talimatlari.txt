=== NW NEON KOCAELI VERITABANI KURULUM TALIMATI ===

Bu talimatlar, komut istemi (CMD) uzerinden NWneonv1.0.bak yedek dosyasindan 'per10Database' veritabanini kurmak (restore) icindir.

1. "NWneonv1.0.bak" dosyasini bilgisayarinizda "C:\Yedek\" klasorune koyun.
   (Eger Yedek klasoru yoksa C surucusunde olusturun)
2. Komut Istemini (CMD) Yonetici Olarak Calistirin.
3. Asagidaki komutu CMD ekranina yapistirip Enter'a basin:

Eger SQL Server surumunuz ucretsiz SQLEXPRESS ise:
sqlcmd -S .\SQLEXPRESS -E -Q "RESTORE DATABASE per10Database FROM DISK='C:\Yedek\NWneonv1.0.bak' WITH REPLACE"

Eger SQL Server tam surum (sadece localhost veya bilgisayar adi) kullaniyorsaniz:
sqlcmd -S localhost -E -Q "RESTORE DATABASE per10Database FROM DISK='C:\Yedek\NWneonv1.0.bak' WITH REPLACE"

--- Onemli Notlar ---
* -S .\SQLEXPRESS : SQL Sunucu adidir. Kocaeli'ndeki bilgisayarin SQL ismine gore burayi degistirmeniz gerekebilir.
* -E : Windows yetkilendirmesi (sifresiz, direkt Windows oturumu ile) giris yapilacagini belirtir.
* WITH REPLACE : Veritabani daha onceden varsa uzerine yazilmasini saglar.
